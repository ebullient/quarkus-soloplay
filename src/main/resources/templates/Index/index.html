How It Works

This application combines:

  • Quarkus - High-performance Java framework
  • LangChain4j - AI integration with prompt engineering
  • Ollama - Local LLM inference (qwen2.5:14b-instruct for chat, nomic-embed-text for embeddings)
  • Neo4j - Graph database for campaign data and vector search
  • Renarde MVC - Web framework for interactive pages

Powered by Quarkus, LangChain4j, Ollama, and Neo4j
